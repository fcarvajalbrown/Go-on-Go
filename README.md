Go-on-Go es una implementación del juego de mesa estratégico Go, escrita en el lenguaje de programación Go. El proyecto incluye la lógica principal del juego y una interfaz básica para jugar partidas locales, explorando estructuras propias del juego como la colocación de piedras, captura de grupos y cálculo de territorio. Está diseñado como una demostración práctica de desarrollo de juegos en Go y como referencia para aprender sobre el lenguaje y su ecosistema.
